# C C B PLUS Beta

基于 [Koikokokokoro/astrbot_plugin_ccb_plus](https://github.com/Koikokokokoro/astrbot_plugin_ccb_plus) 的个人定制增强版。

> Beta 分支为个人需求定制版，功能更激进；如需稳定原版，请使用原作者仓库。

## Beta 版新增内容（共 15 项）

### 权限与群聊控制
1. **群聊白名单**：可限制仅指定群聊使用插件。
2. **群聊单独配置**：使用 `template_list` 为不同群单独设置参数。
3. **群聊每日 CCB 次数限制**：每个群可单独设置每日 `/ccb` 次数上限。
4. **群聊未授权静默返回**：不再刷屏提示“未开启功能”。

### 用户保护与管理
5. **管理员默认加入白名单**：插件会默认把 AstrBot 管理员加入 `white_list`。
6. **机器人自身默认加入白名单**：机器人自身 ID 会自动加入 `white_list`。
7. **`ccbnodo` 管理员指令**：快捷切换指定用户防被 CCB 状态。
8. **`ccbclear` 管理员指令**：清理指定用户的被 CCB 与 CCB 他人记录。

### 暴击与管理员增强
9. **管理员超级暴击倍率**：可配置管理员暴击时使用更高倍率。
10. **超级暴击伪装普通暴击**：触发超级倍率时仍显示普通“暴击”。
11. **管理员额外暴击率**：管理员可获得额外暴击概率。
12. **管理员豁免养胃**：管理员可配置为不触发频率养胃/随机养胃。

### 显示与发送方式
13. **头像显示开关**：可关闭 CCB 结果中的头像图片。
14. **合并转发发送**：支持尝试用合并转发发送 CCB 结果，不支持时自动回退普通消息。
15. **日志配置置顶**：`is_log` 放到配置最上方，方便管理。

## 当前主要指令

| 指令 | 权限 | 说明 |
|---|---|---|
| `/ccb [@目标]` | 所有人 | 执行 CCB，未 @ 默认自己 |
| `/ccbtop` | 所有人 | 被 CCB 次数排行榜 |
| `/ccbvol` | 所有人 | 累计注入量排行榜 |
| `/ccbinfo [@目标]` | 所有人 | 查询用户 CCB 信息 |
| `/ccbmax` | 所有人 | 单次最大注入排行榜 |
| `/xnn` | 所有人 | 小南梁排行榜 |
| `/ccbclear [@目标]` | AstrBot 管理员 | 清除目标 CCB 记录 |
| `/ccbnodo [@目标]` | AstrBot 管理员 | 切换目标防被 CCB 状态 |

## 配置结构亮点

- `is_log`：日志配置置顶。
- `display_settings`：显示设置折叠配置。
- `group_configs`：群聊单独配置，使用 `template_list`。
- `admin_settings`：管理员增强折叠配置。

## 安装

推荐使用本仓库地址安装：

```text
https://github.com/ERX399/astrbot_plugin_ccb_plus_beta
```

如果曾安装原版或旧 beta 版，建议先卸载旧插件目录后再安装，避免目录名/元数据冲突。

- 群聊每日次数统计：`data/ccb_daily_limit.json`
